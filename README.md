Dự án phân tích toàn diện dữ liệu thương mại điện tử từ hệ sinh thái Olist (Brazil) với quy mô hơn 100,000 đơn hàng. Mục tiêu chính là tối ưu hóa tỷ lệ giữ chân khách hàng thông qua mô hình phân khúc RFM và trực quan hóa dòng doanh thu đa chiều.
Công cụ & Kỹ thuật sử dụng
Ngôn ngữ: Python (Pandas, NumPy) để thực hiện quy trình ETL, làm sạch dữ liệu và tính toán chỉ số RFM.

Truy vấn: SQL để hợp nhất các bảng dữ liệu khách hàng, đơn hàng và thanh toán.

Trực quan hóa: Power BI với ngôn ngữ DAX nâng cao để tạo ra các báo cáo tương tác linh hoạt.

Mô hình phân tích: RFM Analysis (Recency, Frequency, Monetary).
